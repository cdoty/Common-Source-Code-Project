ASIEL PECOS Emulator

Supports bios file PECOS_BIOS111.rom (v1.11) or PECOS_V2.00_PRB0127.rom (v2.00) for model PECOS, or PECOS_128A_V400.rom (v4.00) for model PRICK.

Incomplete disk drive support has been implemented. It is able to load the boot sector and subsequently load additional sectors and tracks.
It will load from an inserted raw 360k floppy image, on each model.

I've included Meteor Maid, a prototype game that I'm developing for other TMS-9918a based systems. It is located in the tools/pecos folder.
The .bin file is a module, and the .img is a floppy disk.
